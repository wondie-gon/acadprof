== Description ==

Acadprof is WordPress theme developed for a new web development project of a client. It's adapted to be used for simple blogging websites. Suitable for academic professionals and personal blogging sites. Theme used a renowned WordPress starter theme framework Understrap. Acadprof is still on improvement to include more features and functionalities.

== Installation ==

1. First download from github.
2. Extract the zipped folder in your WordPress at "/wp-content/themes/".
3. In your admin panel, go to Appearance -> Themes and click 'Acadprof' theme.
4. Click on the 'Activate' button to use it right away.
